# Charmify

A habit tracker using [Charm CLI](https://github.com/charmbracelet/) libraries.

![](./tape/charmify.gif)

## Usage

Run the gRPC backend:
```yaml
make run_server
```

Run the habit tracker CLI:
```yaml
make tui
```
